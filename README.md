# Concert-management-system
This is a concert booking system that is made using Python and MySQL

To run this project 

My Application is a concert management system, and it allows a user to create an account, update, and delete the same account. 
Once a user has an account, they can log in and use the application. A user can book a ticket, update and delete it, and simultaneously add a favorite.
Technologies used: Python, MySQL, Tkinter
To have concert details at one location.
run 
python3 firstpage.py 
or for previous versions 
python firstpage.py
